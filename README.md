# 青幻工具箱 — Chrome 插件

AI 翻唱生成工具箱。支持邮箱注册或 API Key 登录，批量生成 Suno 翻唱曲目。

## 功能

- 🎵 上传参考音频，一键生成翻唱（批量多风格）
- 🎶 My Workspace：浏览 Suno 歌单，直接翻唱或下载
- 🔄 任务队列管理，实时进度查看，支持批量下载
- ⬇️ 成品下载：MP3 / WAV / 歌词 / 风格参数

## 安装方法

> 本插件为开发者模式加载，**无需**上架 Chrome 应用商店。

1. 点击本页右上角绿色 **Code** 按钮 → **Download ZIP**
2. 解压 ZIP 文件
3. 打开 Chrome，地址栏输入 `chrome://extensions`
4. 右上角开启 **开发者模式**（Developer mode）
5. 点击 **加载已解压的扩展程序**（Load unpacked）
6. 选择解压后的文件夹

## 使用方法

1. 安装完成后，点击 Chrome 右上角插件图标找到 **青幻工具箱**，固定到工具栏
2. 使用邮箱注册 或 输入购买的 **API Key** 登录
3. 打开 [suno.com](https://suno.com) 任意页面，插件会自动捕获 Cookie
4. 在侧边栏上传音频、配置风格，提交翻唱任务

## 获取额度

- 邮箱注册后初始额度为 0，联系客服开通
- 充值比例：**1元 = 20 工具箱额度**
- 点击插件内「立即充值」即可扫码联系微信

## 常见问题

**Q: 登录失败？**
确认 API Key 正确，或邮箱密码正确。

**Q: Cookie 未绑定 / Suno 积分显示 --？**
登录后打开一次 suno.com，插件自动捕获 Cookie，刷新即可。

**Q: 任务一直处理中？**
Suno 生成需要 1-3 分钟，点任务队列「刷新」按钮查看最新状态。

**Q: 翻唱报错「CAPTCHA 未通过」？**
系统自动处理，稍后重试即可；若持续失败请联系客服。
