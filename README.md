# Suno Cover Tool — Chrome 插件

一键翻唱生成工具。通过 API Key 接入服务，批量生成 Suno 翻唱曲目。

## 功能

- 🎵 上传参考音频，一键生成翻唱
- 🔄 批量任务管理，实时进度查看
- 🍪 自动捕获并绑定 Suno Cookie
- ⬇️ 成品音频一键下载

## 安装方法

> 本插件为开发者模式加载，**无需**上架 Chrome 应用商店。

1. 点击本页右上角绿色 **Code** 按钮 → **Download ZIP**
2. 解压 ZIP 文件，得到 `suno-api-google-main` 文件夹
3. 打开 Chrome，地址栏输入 `chrome://extensions`
4. 右上角开启 **开发者模式**（Developer mode）
5. 点击 **加载已解压的扩展程序**（Load unpacked）
6. 选择解压后的文件夹

## 使用方法

1. 安装完成后，点击 Chrome 右上角扩展图标，找到 **Suno Cover Tool**
2. 输入购买的 **API Key**（格式：`sk-xxxxxxxxxxxxxxxx`）登录
3. 打开 [suno.com](https://suno.com) 任意页面，插件会自动捕获 Cookie
4. 在插件侧边栏上传音频、配置风格，点击生成

## 获取 API Key

请联系客服购买 API Key。

## 常见问题

**Q: 登录失败？**
确认 API Key 正确，且网络可以正常访问。

**Q: Cookie 未绑定？**
登录后访问一次 suno.com，插件会自动捕获。

**Q: 任务一直处理中？**
Suno 生成需要约 1-3 分钟，请耐心等待。
